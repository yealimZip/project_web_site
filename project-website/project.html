AI 기반 스마트 출석 인증 솔루션


프로젝트 목표


AI 기반 얼굴인식 출석 시스템은 기존의 비효율적인 출석 방식 문제를 해결하기 위해 개발되었습니다.
본 프로젝트는 얼굴 인식 기술과 GPS · 시간 기반 검증을 결합하여, 빠르고 정확한 출석 인증 환경을 제공하는 것을 목표로 합니다.

우리는 복잡한 수기 절차나 QR 스캔이 필요없는,
사용자가 개인 휴대폰 카메라로 얼굴 인식을 하면 자동으로 출석이 처리되는 완전 자동화 출석 시스템을 구축하고자 했습니다.


프로젝트 필요성 (Why)


  • 기존 수기 출석의 번거로움
  • 대리 출석 문제
  • 실시간 자동화의 필요성
  • 출석 정확도 향상

이러한 문제를 해결하기 위해,
실시간 얼굴인식 + 위치 인증 + 자동 기록 저장이 가능한 시스템이 필요하다고 생각하여 개발하게 되었습니다.


전체 시스템 구조(Architecture)


• 모바일 앱에서 얼굴 촬영 → 이미지 전처리(dlib, OpenCV)

• 특징 벡터 추출 후 Spring Boot 서버로 전송

• 서버는 등록된 임베딩과 비교하여 출석 여부 결정

• MySQL DB에 기록 저장 후 결과 반환



API 구조


POST /auth/login
{ "id": "user01", "password": "1234" }
POST /face/register
{ "userId": 1, "image": "base64-image" }
POST /attendance/verify
{ "image": "base64", "lat": 37.56, "lng": 127.01 }

각 API는 JWT 기반 인증을 사용하며, 모든 응답은 JSON 형태로 제공됩니다.


주요 기능 상세


1) 얼굴 등록 기능

- dlib을 활용한 얼굴 랜드마크 검출

- 얼굴 영역 크롭 및 정규화

- 사용자의 대표 임베딩 백터 DB 저장


2) 실시간 출석 인증

- 0.9초 내 얼굴 분석 및 매칭

- GPS 기반 강의실 위치 검증

- 시간대 기반 자동 출석 부여


3) OpenCV + dlib을 함께 쓴 이유

OpenCV는 빠르고 안정적인 이미지 전처리(해상도 조절, 노이즈 제거, 얼굴 영역 검출)를 담당하고,
dlib은 128D 얼굴 임베딩 생성과 랜드마크 분석을 통해 높은 정확도의 얼굴 비교를 수행합니다.
정확한 인식률 + 실시간 처리 속도를 모두 충족시키기 위해 두 라이브러리를 함께 사용하여 구현하였습니다.
