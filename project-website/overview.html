프로젝트 소개

AI 기반 출석체크 시스템은 얼굴 인식 기반의 자동 출석 체크 플랫폼으로, 기존 수기 출석 방식의 비효율성과 대리출석 문제를 해결하기 위해 개발되었습니다.

실시간 얼굴 인식 AI, 모바일 앱, 관리자 웹 대시보드를 통합하여 정확하고 편리한 출석 환경을 제공합니다.

FaceOn 로고 의미




FaceOn 로고는 얼굴인식 기술을 기반으로 출석을 보다 쉽게 만든다는 프로젝트의 핵심 가치를 직관적으로 담아 제작하였습니다.

로고의 사각 형태는 얼굴 프레임을, 전자 회로 패턴은 AI 인공지능 기술과 컴퓨터비전 기술을 의미하며, 파란색 미소 라인은 사용자의 편의성과 긍정적인 피드백을 표현합니다.

개발 목표
  • 정확하고 빠른 자동 출석 시스템 구축
  • 대리출석 방지 및 신뢰도 향상
  • 학생·관리자 모두를 고려한 UX 제공
  • 출석 데이터를 DB에 저장하고 시각화하여 관리 효율 증대
  • 실시간 반응 속도를 고려한 AI 모듈 최적화
시스템 구성

본 시스템은 다음과 같은 구조로 구성되었습니다.

  • 모바일 앱 (React Native) – 얼굴 인식 기반 출석
  • 관리자 웹 (HTML) – 출석 현황 조회, 사용자 관리, 출석 기록 시각화
  • Spring Boot 서버 – REST API 제공
  • AI 모듈 (dlib + OpenCV) – 얼굴 인식 및 특징 추출
  • MySQL 데이터베이스 – 출석 기록과 사용자 정보 저장
주요 기능
  • 실시간 얼굴 인식 출석 기능
  • 학생 앱 + 관리자 웹 통합 환경 제공
  • 사용자 등록, 사진 업로드, 얼굴 데이터 관리
  • 출석 로그 자동 저장 및 시각화
  • AI 인식 속도 최적화 및 보안 강화
